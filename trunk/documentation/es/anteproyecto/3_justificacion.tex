\section{Justificación}
{
	Con este proyecto de grado se pretende aplicar el modelo de trabajo colaborativo a la producción de animaciones. Se considera que este modelo funciona bien para este tipo de trabajos, teniendo en cuenta que los gráficos se hace de manera vectorial y que de esta manera se pueden enviar por red los diversos componentes gráficos del proyecto y mediante peticiones al servidor se podría crear progresivamente la jerarquía del proyecto de animación.
	
	Con el uso de la herramienta a desarrollar, las empresas dedicadas a la producción de animaciones tendrían un gran ahorro en el tiempo de producción, lo que implica ahoro de dinero para la organización.  Además el modulo para la Administración que se encargara de la asignación de tareas y llevara una historia del proceso de producción, permitirá a los directores detectar errores en el proceso y ayudara en la toma de decisiones administrativas para el proyecto en curso o futuros proyectos.
	
	Este proyecto resuelve un problema real y latente para la industria de produccion de animaciones,
	implica aspectos relacionados con el desarrollo de software y puede constituir como una herramienta novedosa en su campo de accion.
}
