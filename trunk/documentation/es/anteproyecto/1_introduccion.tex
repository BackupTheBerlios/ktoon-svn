\section{Introducción}
{
	Durante muchos años la industria de producción de animaciones se ha regido por el proceso de animación cuadro-a-cuadro, el cual se trata de dibujar todos los \hyperlink{frame}{cuadros} que componen la animación, recientemente se han incorporado al proceso algunas técnicas avanzadas de \hyperlink{tweening}{interpolación de movimiento} que simplifican el proceso.
	\ds
	Con este proyecto se pretende facilitar la producción de animaciones mediante el enfoque tradicional, así como incrementar los posibles beneficios económicos que pueden generar las ventajas de tiempo que se provee al proceso con este proyecto.
	\ds
	El proyecto esta enfocado directamente a las empresas desarrolladoras de producciones animadas, quienes podrán encontrar en este proyecto una solución muy económica a muchos problemas. Adicionalmente podrá ser utilizado por cualquier persona interesada en la producción de animaciones.
}
