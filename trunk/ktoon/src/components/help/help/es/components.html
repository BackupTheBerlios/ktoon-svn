Componentes Principales

Hace muchos años atrás, cuando dimos inicio a este proyecto y nos encontrabamos diseñando la primera interfaz gráfica para KTooN, escogimos como su primera definición: "Paquete de Herramientas para Animación 2D". En la actualidad, creemos que el concepto se mantiene a pesar de que la interfaz se mantenga en constante evolución.
El primer paso para utilizar KTooN es conocer el ambiente de trabajo, que no es mas que un conjunto de componentes mediante los cuales es posible diseńar, construir y visualizar proyectos de animación 2D.
