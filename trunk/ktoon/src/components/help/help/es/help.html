Módulo de Ayuda

Si estás leyendo este párrafo, entonces te encuentras en el Módulo de Ayuda. Así es, la tercera pestaña del área de trabajo corresponde a un pequeño manual mediante el cual esperamos darte las mínimas pistas para que puedas disfrutar de este programa, creando tus propias animaciones.

Al lado derecho del área de trabajo, podrás apreciar la tabla de contenidos de este manual. Para ir a una sección o título en particular, basta con dar un click sobre el ítem que te interesa.

Fig #1. Interfaz Módulo de Ayuda KTooN

TIP: Cada vez que desees consultar el Módulo de Ayuda, sólo debes presionar la tecla F11.

Recuerda que si tienes dudas o inquietudes que no resuelva este manual, puedes buscar más ayuda en la página web del proyecto http://www.ktoon.net. Allí podrás interactuar con otros usuarios y con el equipo de desarrollo de KTooN. Junto a nuestra comunidad, con tus aportes, ideas y sugerencias podemos convertir este proyecto en una herramienta profesional libre que permita el nacimiento de millones de obras de arte animadas!
